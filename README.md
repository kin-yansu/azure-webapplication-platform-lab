# クラウドネイティブWebアプリケーションの検証環境

## 概要

このリポジトリは、FastAPIを中心としたクラウドネイティブWebアプリケーションの検証環境です。

Azure App Serviceへのデプロイ、コンテナ化、Infrastructure as Code (IaC)、非同期メッセージングなどを実践しながら学習・検証することを目的としています。

## Scope

✔ FastAPI
✔ Docker
✔ Azure App Service
✔ Azure OpenAI
✔ Azure AI Search
✔ Azure Storage
✔ Azure Service Bus / Pub/Sub
✔ IaC (Bicep)

## Future

- APIM
- Application Gateway
- GitHub Actions
- Monitoring


## Design Principles

- Infrastructure as Code first
- Container first
- Cloud-native architecture
- Managed Identity over secrets
- Private Endpoint whenever possible
- Reproducible development environment
